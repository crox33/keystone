# playground/

Throwaway scratch space. Copy `example_idea.py`, hack, press Debug.

- Not production, not imported by `src/`, not held to coverage.
- Files here other than `example_idea.py` and this README are git-ignored
  (see `.gitignore`) — experiment without polluting the repo.
- Proven ideas get promoted into `src/keystone/` with tests + reconciliation.
